# 干瞪眼记分

一个手机端为主的**干瞪眼**记分应用。单个 `index.html`，无需后端、无需安装，浏览器打开即用，数据存在本地（IndexedDB + localStorage）。

## 功能

- **总分榜**：自动排名、领先者高亮，等宽数字编辑式排版
- **记一局（计算器）**：选赢家 → 给每个输家点剩牌 / 2 的个数 / 炸弹个数，自动算分，赢家收走输家之和（零和）
- **对局存档**：开新对局、回看/切换/重命名/删除以前任意一盘
- **战绩 · 走势**：累计分折线图（每人一条线，可单独高亮）、本场之最、每人战绩表
- **导出 / 导入备份**：JSON 文件备份，换设备搬数据
- 浅色 / 深色自动跟随系统，尊重 `prefers-reduced-motion`

## 算分规则

每个输家：

```
得分 = 剩牌数 × 2^(2的个数) × 2^(炸弹个数) × (剩牌≥5 关到底 ×2) × 2^(场上炸弹数)
得分 = min(得分, 100)
```

赢家收走所有输家之和（零和结算）。三张及以上算炸弹、2 不入顺、能拆炸弹。

也支持每局直接记录各人加减分（导入历史数据用），与计算器记法共存。

## 使用

直接用浏览器打开 `index.html` 即可。
